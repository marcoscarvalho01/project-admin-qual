DataAnnotation Project Midas — Admin Qualification

Admin Qualification Deliverables

This page contains the interactive form demos submitted as part of the Project Midas — Admin Qualification Task. Each page is a functional mockup of how the task workflow would look on the platform, built to illustrate the proposed form design and user experience.

Qualification Exam Form Workers Selection

Mockup of the qualification exam shown to worker candidates. Covers all three question types: simple retrieval (Cat. A), single-doc complex reasoning (Cat. B), and multi-document prompt creation (Cat. C). Uses a resizable side-by-side layout with instructions on the left and the form on the right.

Main Task Form Cat. A & B

Mockup of the production task form for Categories A and B (single-file questions). Demonstrates the step-by-step progressive disclosure workflow — each step is revealed only after the worker confirms the previous one — along with the SEC filing URL input, PDF upload, prompt writing, answer, supporting facts, and rationale fields.

Multi-Document Task Form Cat. C

Mockup of the production task form for Category C (multi-file complex questions). Workers are required to synthesize information across up to 40 SEC filings to write a single complex prompt, provide a perfect answer, map supporting facts to each source document, and write a detailed cross-document rationale.

Rate & Review Form Quality Assurance

Mockup of the Rate & Review (R&R) task assigned to Tier 1 reviewers during the QA phase. Features a collapsible review instructions panel, an embedded side-by-side view of the original worker submission (editable), and a structured evaluation form where reviewers rate the initial and final submission quality, flag error categories, and provide written feedback.
